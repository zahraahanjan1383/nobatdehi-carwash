## توضیح کلی پروژه
 سیستم نوبت‌دهی کارواش یک ابزار دیجیتال برای مدیریت صف مشتریان و زمان‌بندی خدمات است. این سیستم باعث می‌شود مشتریان بدون معطلی طولانی، خدمات خود را رزرو کنند و کارواش بتواند ظرفیت و نیروی انسانی را بهتر مدیریت کن
